# duplo-train
A webapp to control my sons Duplo train. Built with Nuxt.Js, Vuex and Vuetify.

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# generate static project
$ yarn generate
```
